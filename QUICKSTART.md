# 快速開始指南 🚀

## 5 分鐘上手晶晶體生成器

### 📦 安裝

```bash
# 1. 安裝依賴
pip install -r requirements.txt

# 2. 下載模型（約 3GB，需要 3-5 分鐘）
python scripts/download_models.py

# 3. 啟動應用
./run.sh
```

應用會自動在瀏覽器中打開 http://localhost:8501

---

## 💡 使用方法

### 基本使用流程

1. **輸入主題**
   - 在主介面的文字框中輸入任何主題
   - 例如：「週末計畫」、「咖啡廳閒聊」、「健身房」

2. **點擊生成**
   - 點擊「✨ 生成晶晶體」按鈕
   - 等待 5-15 秒生成文字

3. **播放語音**
   - 自動生成語音檔案
   - 點擊播放器聆聽
   - 可下載 MP3 檔案

### 進階設定

在左側側邊欄可以調整：

#### 文字生成參數
- **最大長度** (50-300): 文字長度
- **創意程度** (0.5-1.5): 越高越有創意但可能不連貫
- **多樣性** (0.5-1.0): 輸出的變化程度

#### 語音參數
- **慢速語音**: 更清晰但較慢的語音

#### 音訊管理
- 查看暫存檔案數量和大小
- 一鍵清理舊檔案

## 🎯 常見問題

### Q: 生成的文字英文比例不夠高？
**A**: 已經優化 prompt 讓英文比例達到 60-70%。如果還不滿意，可以：
- 多試幾次
- 提高「創意程度」參數
- 使用更具體的主題

### Q: 語音聽起來不自然？
**A**: gTTS 對混合語言的支持有限。建議：
- 使用「慢速語音」選項
- 混合語言會優先使用中文語音引擎

### Q: 生成時間太長？
**A**: 正常生成時間約 5-15 秒。如果更長：
- 降低「最大長度」參數
- 檢查系統資源使用情況
- 重啟應用

### Q: 如何下載音訊？
**A**: 生成語音後，點擊「📥 下載音訊檔案」按鈕即可。
